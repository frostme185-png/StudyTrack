How to test auto-tracking
1. Make sure your website is added in the Websites tab
2. Open your browser and go to that site (e.g. coursera.com)
3. Keep that browser window focused/in front
4. Watch Active window above — it should show the page title
5. Matched site should change from "None" to the site name
6. Every 2 seconds the accum counter ticks up
7. Every 30s (or when you switch sites) data saves to disk

⚠ Common issues:
• If Active window shows empty → run app as Administrator
• If title doesn't contain domain keyword → edit site domain to match title text
• Idle pause kicks in after 5 min of no keyboard/mouse input
